# take_home_task2

### How To Run

Run Flask API:

```sh
$ cd flask_api
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ python run.py
```

Run Svelte Front-end:

```sh
$ cd svelte_dashie
$ npm install
$ npm run dev
```

Test at [http://localhost:8080/](http://localhost:8080/)
